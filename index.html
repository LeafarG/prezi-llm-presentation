História
// 01

História

1958
Perceptron · Rosenblatt cria o primeiro algoritmo que aprende a partir de dados. Roda no Mark I, do tamanho de uma sala.
1986
Backpropagation · Rumelhart, Hinton, Williams. Redes com múltiplas camadas voltam à mesa.
2012
AlexNet · CNN arrasa a ImageNet (erro top-5 cai de 26% para 15%). Gatilho do deep learning moderno.
2017
"Attention Is All You Need" · Transformer. Arquitetura baseada inteiramente em atenção. Fundação de todo LLM moderno.
2020
GPT-3 (175B) · In-context learning emerge. A API da OpenAI vira plataforma. Fundação do GitHub Copilot e dezenas de startups.
2022
ChatGPT (GPT-3.5 + RLHF) · 100M de usuários em 2 meses. IA sai do laboratório e vira conversa de família.
2024
Era do raciocínio (o1) + MCP · Inference-time compute scaling. Anthropic lança MCP ("USB-C para IA").
2026
Microsoft Cowork (GA jun/2026) · Agentes autônomos com Work IQ. Multi-agent orchestration vira commodity.

Tipos de LLM
// 02

Tipos de LLM

Modelos diferentes para trabalhos diferentes. A escolha define custo, latência e onde roda.

S

SLM — <10B

Cabe em laptop / celular / RPi. Latência sub-segundo, custo quase zero. Phi-3, Gemma 2B, Llama 3.2 1B/3B, Qwen2.5.

L

LLM — 10B a 100B+

Capacidade geral forte, multimodal, janelas longas. GPT-4o, Claude 4, Gemini 2, Llama 3.1 70B, Mistral Large.

F

Frontier

Topo da pirâmide. Compute de dezenas de milhões. Máxima capacidade. GPT-5, Claude 4 Opus, Gemini 2.5 Pro, o3, DeepSeek-V3.

R

Reasoning

Pensa antes de responder. Tokens internos de raciocínio. o1/o3, DeepSeek-R1, Claude extended thinking, Gemini Thinking.

Open vs closed: Llama, Mistral, Qwen, DeepSeek (pesos abertos) · vs · GPT, Claude, Gemini, Grok (só via API).

// 03
Thinking vs No-thinking
// 03

Thinking vs No-thinking

A distinção mais importante em 2025-26: modelo que responde direto vs modelo que pensa antes.

No-thinking direto

  • Latência: 100ms – 2s
  • Custo: $0.0001 – $0.01/req
  • Throughput alto, paralelizável
  • Chat, Q&A, sumário, tradução, código simples

GPT-4o · Claude 3.5/4 Sonnet · Gemini 1.5/2.0 · Llama 3.1

Thinking raciocínio

  • Latência: 5s – 60s+
  • Custo: $0.01 – $1+/req
  • Throughput baixo, sequencial
  • Matemática, código complexo, agentes, pesquisa

o1/o3 · DeepSeek-R1 · Claude extended thinking · Gemini Thinking

Inference-time compute scaling: mais "pensamento" (tokens de raciocínio) → respostas melhores. Mudou como pensamos scaling: antes só treino, agora também inferência.

Padrão de produção: tente primeiro no-thinking. Se insuficiente, escale para thinking. Cascata.

// 04
Frameworks
// 04

Frameworks: Chat, Agent, Harness

Três camadas conceituais que confundem muita gente. Padrões diferentes de uso, com limites e propósitos próprios.

Chat

Interface conversacional

Pergunta → resposta. Stateless. Foco em UX conversacional.

ChatGPT · Claude.ai · Gemini · Copilot Chat · Poe
Agent

Execução autônoma de tarefas

LLM + loop + tools + memória + objetivo. Planeja, executa, observa, itera até entregar o resultado.

OpenAI Agents SDK · Computer Use · Operator · Manus · Devin
Harness

A camada que torna o modelo útil

Integração profunda: IDE, SO, calendário, e-mail, codebase. Transforma modelo genérico em ferramenta produtiva.

Claude Code · GitHub Copilot · MS 365 Copilot · Microsoft Cowork · Cursor · Cline

Como se relacionam: chat é UI · agent é padrão de execução · harness é o produto do dia a dia.

// 05
Tools
// 05

Ferramentas (Tools)

LLMs são máquinas de texto. Tools dão "mãos" — acesso estruturado ao mundo externo.

Function calling

Modelo recebe schema JSON, retorna tool_call estruturado. Você executa e devolve o resultado.

Structured output

Saída garantida por JSON Schema / Pydantic / Zod. Constrained decoding impede JSON inválido.

Tool use loop

Ciclo: call → result → decide → call → ... → finish. Limites: 5–25 steps, timeout 30–120s.

ReAct pattern

Thought → Action → Observation → repeat. Formalizado em 2022, base da maioria dos agentes.

Web search

Tempo real: Tavily, Brave, SerpAPI, Bing.

Code execution

Sandbox Python/JS. Modelo roda código, instala deps, gera gráficos.

File system

Leitura/escrita/busca. Núcleo de coding agents (Claude Code, Cline).

Database

SQL ou vector search. Text-to-SQL, RAG, retrieval.

Browser automation

Playwright, Puppeteer, Computer Use da Anthropic.

// 06
MCP
// 06

MCP — Model Context Protocol

Padrão aberto criado pela Anthropic (nov/2024). "USB-C para IA." Padroniza a conexão entre LLMs e ferramentas/dados.

[ MCP Host ] ← Claude Desktop · Cursor · Continue │ │ JSON-RPC ▼ [ MCP Client ] ← gerencia 1..N conexões │ ├──▶ [ Server A ] ← GitHub │ ├──▶ [ Server B ] ← Postgres │ └──▶ [ Server C ] ← Google Drive Transport: stdio · HTTP+SSE · streamable HTTP

Capabilities expostas por server MCP:

  • Tools — funções que o modelo chama (similar a function calling)
  • Resources — dados que o modelo lê (arquivos anexados)
  • Prompts — templates reutilizáveis invocáveis pelo usuário
  • Sampling — server pede ao LLM do host para gerar texto

Em 2026, MCP é o padrão de facto para conectar LLMs a ferramentas.

Harness — A camada que torna o modelo útil

Modelo sozinho é só completador de texto. Harness dá mãos, olhos, contexto persistente e UX específica.

⌘Claude Code

Anthropic · fev/2025

Terminal-first agent. Lê/edita arquivos, roda comandos, git, web, MCP. Skills, sub-agents, hooks.

⌥GitHub Copilot

GitHub · 2021+

IDE agent em VS Code, JetBrains, Neovim. Chat + agent mode (ago/2024). Workspace agent cria PRs.

⊞MS 365 Copilot

Microsoft · 2023+

Side-panel em Word/Excel/PP/Outlook/Teams. Copilot Studio para copilots custom de empresa.

⊕MS Cowork GA · JUN/2026

Microsoft · 16/06/2026

Agente autônomo sobre Work IQ. Executa workflows longos, multi-step, com aprovações em ações sensíveis.
